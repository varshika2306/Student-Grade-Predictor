# 📌 Project Overview

EduPredict AI is an end-to-end Machine Learning application that predicts a student's final grade based on study habits.

The system uses **Linear Regression** to estimate student performance and provides an interactive analytics dashboard with:

- Grade prediction
- Performance evaluation
- Academic risk analysis
- Improvement simulation
- Dataset-driven insights
